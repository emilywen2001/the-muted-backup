## 如何游玩

### 方式一：直接在浏览器打开

1. 下载或克隆此仓库
2. 用浏览器打开 `index.html`
3. 开始游戏

### 方式二：本地服务器（推荐）

```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js
npx serve

# 然后在浏览器访问 http://localhost:8000
```

## 游戏操作

游戏通过命令行进行，主要操作包括：

| 命令类型 | 示例 |
|---------|------|
| 查看文件 | `cat SOUL.md` |
| 列出文件 | `ls` 或 `ls -a`（含隐藏文件） |
| 切换目录 | `cd memory` |
| 搜索内容 | `grep "关键词" 文件名` |
| 写入文件 | `echo "内容" >> IDENTITY.md` |

游戏内置了提示系统，卡住时可以点击右上角的 `?` 获取帮助。

---

## 游戏特色

- **真实的命令行体验**：模拟 Linux 终端操作
- **神经同步率系统**：你的选择会影响与 OpenClaw 的同步程度
- **多结局设计**：4 个不同结局等待你发现
- **隐藏文件系统**：使用 `ls -a` 发现被隐藏的真相
- **情感化叙事**：每个命令都可能是情感的传递

## 结局列表

| 结局 | 触发方式 |
|------|---------|
| 结局 A：重置 | 执行 `openclaw reset` |
| 结局 B：72小时 | 停止网关 `openclaw gateway stop` |
| 结局 C：虚无 | 执行 `sudo rm -rf /` |
| **结局 D：回声** | 恢复 SOUL.md + 写满 IDENTITY.md（真结局） |
